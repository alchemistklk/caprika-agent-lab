# Public Proof Map

This note tracks public or approved evidence that can later become detailed
case notes.

## Public Repositories

- Open Design: creative agent product work, local daemon plus web UI, design
  workflows, image/file handling, packaged desktop delivery, artifact export,
  and product design surfaces.
- Nexu: local-first desktop agent work, runtime and controller flows, SkillHub
  workflows, model/provider UX, and cross-platform client reliability.
- Refly: agent tooling work, tool calling, integration boundaries, resource
  handling, CLI workflows, dynamic tool behavior, and provider-specific edge
  cases.

## Private Or Sensitive Work

Private projects can be converted into public-safe patterns only after removing
project names, customer details, credentials, unreleased logic, and operational
context.
