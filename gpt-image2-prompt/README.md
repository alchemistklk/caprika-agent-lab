# GPT Image 2 Prompt Library

Reusable prompt workflows for product posters and profile headers.

This folder keeps the prompts generic first. Project-specific details such as
Open Design, 80K stars, maintainer identity, and brand colors are treated as
replaceable variables or example presets.

## Entries

| File | Use |
|---|---|
| `general-poster.md` | General social poster workflow for milestones, launches, release notes, and proof-of-work posts. |
| `general-profile-header.md` | General X/Twitter profile header workflow for personal or project positioning. |
| `style-system.md` | Shared visual constraints and reusable style variables. |

## Prompt Structure

Each workflow should separate:

- `Purpose`: what the image is meant to do.
- `Variables`: project, milestone, audience, palette, objects, copy, and metrics.
- `Composition`: how the image is arranged.
- `Reusable prompt`: the model-ready prompt template.
- `Example preset`: a concrete filled-in version from a real case.
- `Iteration notes`: decisions from prior exploration.

## Current Example Preset

The current filled-in preset is Open Design:

- Header: Open Design maintainer identity.
